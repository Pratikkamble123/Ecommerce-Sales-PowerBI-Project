# E-Commerce Sales Analysis using Power BI

## Project Overview
This project analyzes e-commerce sales data using Power BI to identify
sales trends, top-performing products, and regional sales distribution.

## Tools Used
- Microsoft Power BI
- Excel Dataset
- Power Query

## Dashboard Insights
- Category-wise sales performance
- Monthly sales trend
- Region-wise sales distribution
- Top selling products

## Dataset Source
Kaggle - Sample Superstore Dataset
